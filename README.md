### 1. Clone the Repository

```bash
git clone https://github.com/tsingha97/my-shopping-cart.git
cd my-shopping-cart
```

### 2. Install Dependencies

Using npm:

```bash
npm install
```

Or using Yarn:

```bash
yarn install
```

### 3. Start the Development Server

Using npm:

```bash
npm run dev
```

Or using Yarn:

```bash
yarn dev
```

Visit [http://localhost:5173](http://localhost:5173) to view the app.
